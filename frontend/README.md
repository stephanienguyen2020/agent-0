# Agent Zero — Frontend

Next.js 15 App Router + Tailwind. Set `NEXT_PUBLIC_API_URL` to the FastAPI origin (default `http://localhost:8000`).

For any frontend or UI change, read [`design.md`](design.md) first and use it as the source of truth for visual design decisions.

```bash
npm install
npm run dev
```

RainbowKit, wagmi, World ID IDKit, and SIWE are next integration steps per `docs/09-frontend-specification.md`.
